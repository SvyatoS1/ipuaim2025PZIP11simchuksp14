# GraphDetector: Класифікатор Графіків на TensorFlow.js
GraphDetector: Класифікатор Графіків на TensorFlow.js
GraphDetector — це "клієнтський" веб-додаток, який використовує попередньо навчену нейронну мережу для класифікації зображень математичних графіків у реальному часі. Додаток може розрізнити лінійні, квадратичні та кубічні функції, завантажені користувачем.

## Основний функціонал
- **Класифікація в Реальному Часі**: Аналіз зображень відбувається миттєво у браузері користувача.

- **Інтерактивний UI**: Сучасний інтерфейс з підтримкою Drag & Drop для легкого завантаження файлів.

- **Асинхронне Завантаження**: Модель завантажується у фоновому режимі, не блокуючи інтерфейс.

- **Повний Звіт**: Виводить відсоткову ймовірність для всіх трьох класів (Linear, Parabola, Cubic).

- **Бенчмаркінг**: Вбудоване вимірювання часу завантаження моделі та часу прогнозування.

- **Системні Сповіщення**: Використовує Notification API для сповіщення про завершення аналізу.

## Як Запустити Локально
Для роботи проекту потрібен локальний веб-сервер (через політику безпеки браузера CORS, яка блокує fetch запити для file:/// шляхів).
1. Клонуйте репозиторій:
   ```
   git clone https://github.com/SvyatoS1/ipuaim2025PZIP11simchuksp14.git
   cd ipuaim2025PZIP11simchuksp14
2. Переконайтеся, що у вас є папка model/ з файлами .json та .bin.
3. Якщо у вас VS Code, встановіть розширення Live Server.
   ```
   ext install ritwickdey.liveserver
   ```
4. Клацніть правою кнопкою миші на index.html та оберіть "Open with Live Server".
